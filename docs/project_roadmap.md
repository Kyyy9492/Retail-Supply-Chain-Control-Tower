# Project Roadmap

## Day 1 — Framework Setup

- Create GitHub repository structure.
- Add README, data dictionary, project roadmap, and business question documents.
- Create SQL script templates.
- Create Python module templates.
- Create notebook templates.
- Prepare local setup instructions.

## Day 2 — Local Deployment

- Create virtual environment.
- Install requirements.
- Download Olist and M5 datasets.
- Place raw CSV files into `data/raw/olist/` and `data/raw/m5/`.
- Create MySQL database.
- Run SQL create-table script.
- Import Olist CSV files.
- Run initial KPI SQL queries.
- Open Jupyter notebooks and verify paths.

## Version 1 — GitHub-Ready Core Project

Focus: Olist fulfillment analytics.

Deliverables:

- SQL schema and KPI queries;
- delivery performance analysis;
- seller scorecard;
- Power BI dashboard screenshots;
- README with early findings.

## Version 2 — Higher-Value Analytics Layer

Deliverables:

## Version 3 — Advanced Portfolio Layer

Focus: predictive operations model.

Deliverables:

- late delivery risk model;
- feature importance analysis;
- risk-based operational recommendations;
